Crear un branch

 

  1. Crear un proyecto
  2. Descargar el proyecto en un computador
  3. Modificar archivos desde el computador y subirlos a github
  4. Descargar Archivos modificados al computador
  5. Hacer un fork a un proyecto ya existente
  6. Modificar el fork del proyecto existente desde el comptuador y hacer un Pull Request para que sea incluído en el fork
  7. Aceptar un pull request
  8. Actualizar un fork cuando el proyecto upstream (el original) ha cambiado
  9. Crear un branch
  10. Subir modificaciones a un branch del proyecto
  11. Actualizar un branch usando otro
  12. Unir dos branches
      


Para crear una branch en el repositorio de github, solo tienes que
crearla localmente y despues hacerle push.

--- listar branch
$ git branch
* master

--- crear branch e inmediatamente cambiarse a ella
$ git checkout -b fix_readme
Switched to a new branch "fix_readme"

--- listar branch
$ git branch
* fix_readme
master

--- subir branch a github
$ git push origin fix_readme
Total 0 (delta 0), reused 0 (delta 0)
To g...@github.com:b3t0/JFP.git
* [new branch] fix_readme -> fix_readme

--- listar origen
$ git remote -v
origin g...@github.com:b3t0/JFP.git
